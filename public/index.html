Itinerario da iCal

Versione pensata per Comandi Rapidi su iPhone: scegli il file .ics e genera una pagina finale pronta da salvare o condividere.

Come usarlo in Comandi Rapidi
  1. Carica questo HTML in Safari o in una web view.
  2. Il comando rapido passa il file .ics al browser/script.
  3. Premi il pulsante di esportazione per ottenere la pagina finale.
